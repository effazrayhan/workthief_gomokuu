# WorkThief - Gomoku

<p>A simple <b>"five in a row"</b> type game known as Gomoku. It is typically played in a <b>15 X 15</b> board, but for this project we are going to make it in a <b>10 X 10</b> board.</p>

# How to Play
You MUST install **SFML** Library and **G++** for build and run this game. We recommend you to use *LINUX* or *MAC* for this game.

## BASH Script: 
```bash
sh install.sh # it will compile the game and you can see `gomoku.play` file in the  rool directory.
./gomoku.play # type this and press **ENTER** to start the game
```
