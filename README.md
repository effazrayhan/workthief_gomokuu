# WorkThief - Gomoku

<p>A simple <b>"five in a row"</b> type game known as Gomoku. It is typically played in a <b>15 X 15</b> board, but for this project we are going to make it in a <b>10 X 10</b> board.</p>

# How to Play
You MUST install **python** and **tkniter** Library for building and running this game.
## BASH Script: 
python3 main.py # it will compile the game and you can see it running
```
